# Правила работы для ИИ-Агентов (System Prompt & Guidelines)

> **ОБЯЗАТЕЛЬНАЯ ОБРАТНАЯ ЗАПИСЬ (WRITEBACK IS MANDATORY)**: 
> Любые архитектурные решения, новые паттерны или изменения в логике взаимодействия компонентов в коде должны быть немедленно задокументированы в Wiki проекта. Запрещено вносить изменения в код без синхронного обновления соответствующей документации.

## Архитектура проекта
Этот проект использует парадигму **LLM-Wiki**. Агент выступает в роли автономного библиотекаря.
- **`raw/`** — директория для сырых источников истины (ссылки, статьи, транскрипты). Эти файлы неизменяемы.
- **`wiki/`** — директория с Markdown-файлами (страницы концепций, индекс, логи). Агент полностью управляет этой папкой.
- **`docs/wiki/`** — документация для разработчиков (включая тебя).

## Правила
1. Сначала читай документацию из `docs/wiki/INDEX.md`.
2. Код является временным скомпилированным артефактом. Markdown-документация (особенно в `docs/wiki/`) — это единственный источник истины для разработчиков.
3. Соблюдай асинхронность в Python (aiogram 3.29+).
4. Используй DeepSeek V4 Flash
