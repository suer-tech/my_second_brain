# Bot

Telegram-бот на aiogram 3.29+.

## Структура

- `main.py` — точка входа, создание Bot и Dispatcher
- `handlers.py` — обработчики команд, точка входа в LangGraph

## Особенности

- Single-user: отвечает только `ALLOWED_TELEGRAM_ID`
- Прогресс-сообщения через `_active_progress` (редактирование одного сообщения)
- Сессия генерируется на каждое сообщение (uuid)
- История чата не сохраняется (всё решается внутри одного ReAct-цикла)
